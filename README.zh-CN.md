## 开发
#### :collision: &nbsp; `千万不要将token提交到github上`
```
// 设置 NPM_AUTH_TOKEN 本地创建sh文件用来保存命令(不要提交文件)
export NPM_AUTH_TOKEN=*[你的npm token]*

// 安装依赖包
$ yarn

// 运行项目
$ make local

// 更新版本号
$ make patch // 1.0.X
$ make minor // 1.X.0
$ make major // X.0.0
```
